# english_tree_arrangement

Создать проект в PyCharm, чтобы он сам создал виртуальную среду.

В терминале под виртуальной средой:

git clone https://github.com/krasovskiyhst/pilotpro_parsers.git

pip install -r requirements.txt


Затем в файле main.py, в конце есть строка if __name__ == '__main__':. 
Там можно в PyCharm запустить кнопкой слева "Run main". Или на верхней панели справа.
